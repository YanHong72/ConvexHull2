(a)main.py為我們主要執行程式的地方
(b)defineFunc是放我們自己定義的函式的地方
(c)如果要跑main.py前請先在python安裝好下列模組:
    (1)numpy
    (2)scipy 
    (3)fractions
(d)如果不想跑main.py, 這邊有一個exe執行檔, 可以直接執行這一個程式
如果要執行，可以直接點擊exe檔，或是打開命令字元提示，移動到此資料
夾，然後打main.exe
(e)這裡有一個輸入的範例in.txt，如果想要使用這個範例的話可以打開命
令字元提示，移動到此資料夾，然後打main.exe < in\檔名.txt就可以執行輸入的範例。

以in_5_1.txt為例:
input: main.exe < in\in_5_1.txt 

output: 結果: p在convex hull的裡面
